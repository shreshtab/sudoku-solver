# sudoku-solver

## Installation Instructions

```
// Clone without SSH
git clone https://github.com/shreshtab/sudoku-solver.git

// Clone with SSH
git clone git@github.com:shreshtab/sudoku-solver.git

// Navigate to client directory
cd sudoku

// Install Dependencies
yarn

// Start Application
yarn start
````
